Always up-to-date

Sensitive data such as grades, attendance, and discpline are always available for viewing by parents and admins with changes to data being reflected immediately.

Modern responsive design

Teachers and parents can be on-the-go and student records should be able to travel with them. Our mobile-first approach makes content look great on any device.

Feature rich

Powerful tools are included right out of the box, allowing for the system to management itself.

Application Features

A good SIS connects staff directly to a student's family

Powerful Course Management Tools

School admins can easily create new course and assign instructors to them using preset course catalog templates
Live Attendance

Attendance data logged by instructors is automatically reported to families and admin personnel providing instant peace of mind
Seamless Reporting & Analytics

Report cards, discipline reports, attendance records, and transcripts can all be generated and exported within the blink of an eye
Flexible Gradebook

Instructors have the power to adjust their gradebook exactly to their liking with grade changes being reflected instantly on grade reports and previews
Magic Schedule Builder

A powerful scheduling tool takes the guesswork out of creating a student's schedule with course suggestions and conflicts being highlighted automatically
Discipline Tracking

Logging behavior, both good and bad, is easy to input for instructors & admins with parents being notified instantly
What technologies are used?

Mainframe was written using HTML, CSS, Javascript, and C#. Specifically, it uses ASP.NET Core MVC for managing the database models, controller logic, and frontend views.

How is the data stored?

The application utilizes Entity Framework to interface with the Microsoft SQL Server backend database.
