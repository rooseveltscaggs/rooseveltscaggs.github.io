My Housing Upgraded

Putting the , allocation, and selection of housing resources at your fingertips

Launch Demo

Always up-to-date

Housing data is automatically refreshed, giving students the most accurate information on housing availability without any manual work on the part of housing staff.

Modern responsive design

Designed with a mobile-focused approach, every aspect of the app looks and functions great on any device!

Feature rich

Powerful tools for both admins and end-users are provided, allowing for the system to help manage itself.

Application Features

Housing doesn't have to be complicated

Live Housing Search

Students can always access a live list of housing that is currently available, making their search for suitable housing quick and easy
Live Room Selection

During the room selection process, students can see the status of each of their drawmates live as the selection happens
Easy Application Process

Housing applications are easy to configure and fill out, taking mere seconds to be completed by students
Simplified Housing Lottery

Configuration of student housing priority weights and assignment of randomized housing draw times can be setup and published within minutes
Powerful Admin Tools

Managing housing inventory is a breeze with the ability to add, reserve, and manually assign housing with just two clicks

Automatic Housing Reservations

Once rooms have been selected by students, they are automatically marked as reserved. Thus eliminating the risk of overbooked housing.
What technologies are used?

My Housing Upgraded was written using HTML, CSS, Javascript, and Python. Specifically, it uses the jQuery library and Flask Python framework along with Websockets to allow for live updates.

How is the data stored?

The application utilizes SQLAlchemy in order to interface with the PostgreSQL database, however any database compatible with SQLAlchemy can be configured for use with the app.
